Sales App Demo

Sales is a cross-platform mobile application developed with Xamarin.Forms using the MVVM pattern and Service Locator architecture. It connects to a RESTful API built with .NET and C#, providing a smooth and responsive experience across Android, iOS, and UWP.
The app enables users to sell and manage items categorized by product type, track orders, customers, and sales performance, and supports multiple languages for a global user base.
